Puppet MongoDB

Requirements

  • Puppetlabs apt module or
  • Camptocamp apt module

Example usage

Install MongoDB

node /box/ {
  include mongodb
}

Manage repository

node /box/ {
  class { 'mongodb':
    manage_repo => true,
  }
}

Configure MongoDB

node /box/ {
  class { 'mongodb':
    authentication => false,
    listen_port    => '27017',
    quota          => false,
  }
}
